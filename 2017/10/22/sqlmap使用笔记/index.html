sqlmap使用笔记

首先，sqlmap是一款开源自动化数据库注入软件，基于布尔注入，错误返回注入，时间延迟注入，堆注入，联合注入等技术可以检测操作系统和数据库版本并且从数据库获得控制权和信息，本文介绍一下简单的sqlmap使用命令。

针对入门学者，可以使用

  1. option = [–users –password –dbs –tables –columns –dump]
  2. sqlmap -hh / –version / –update
  3. 参数：–param-del

当GET或POST的数据需要用其他字符分割测试参数的时候需要用到此参数。

例子：

python sqlmap.py -u “http://www.target.com/vuln.php“ –data=”query=foobar;id=1” –param-del=”;” -f –banner –dbs –users
