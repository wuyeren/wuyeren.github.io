wifi无线渗透

这篇文章涉及的是wifi渗透测试所使用的一些指令和方法,首先回顾一下wifi的基本概念，wifi是无线通信的基于802.11系列标准的通信协议，俗称无线局域网，wifi通信以帧的方式进行，按照头部结构的类型字段0, 1, 2可分为三种帧类型，管理帧，负责维护接入点和无线客户端的通信；控制帧，负责接入点和无线客户端的数据交换；数据帧，传输真实的数据。
无线网卡有四种工作模式，managed模式，负责无线客户端与接入点进行接入连接；master模式，无线网卡作为接入点使用的一种工作模式；ad hoc模式，无线客户端使用对等网络的一种工作模式，此时无线无卡兼具接入点和客户端的责任；monitor模式，主要用于无线网络流量监控和嗅探。

  1. 配置无线网卡：

    开启/关闭无线网卡 ifconfig <name> up/down
    查看无线网卡状态 ifconfig <name>
    设置监控模式 airmon-ng start <name>
    指定无线网卡工作频道 iwconfig mon0 channel <num>

  2. 无线网络探测连接

探测无线网络 iwlist <name> scanning
连接无线网络 iwconfig <name> essid 'essid name'
查看路由 route -n
查看arp记录 arp -a
嗅探接入点 airodump-ng mon0
嗅探指定接入点 airodump-ng --bssid <mac> mon0

  1. 数据包注入和捕获

注入数据包 aireplay-ng -9 -e <essid> -a <mac> mon0
隐藏ssid只是接入点广播信标帧ssid字段为空。
解除验证攻击 aireplay-ng -0 <num> -a <mac> --ignore-negative mon0
改变mac地址用于mac地址突破 macchanger <mac> <name>
抓取wep密钥流 airodump-ng -c <channel> -b <bssid> -w keystream
伪造共享密钥验证 aireplay-ng -1 0 -e <essid> -a <mac> -y keystream01-00-21-91-D2-8E-25.xor -h
